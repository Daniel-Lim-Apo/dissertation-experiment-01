# Vector API Service

## Overview

FastAPI service providing vectorization and Qdrant integration for text data.

## Purpose

- Generates text embeddings
- Manages Qdrant vector database operations
- Provides API for vector search and storage

## Files

- **main.py**: FastAPI application
- **embedding.py**: Embedding generation logic
- **Dockerfile**: Container definition
- **requirements.txt**: Dependencies

## Dependencies

- Qdrant client
- FastAPI
- Embedding models
